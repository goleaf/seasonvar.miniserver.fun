# Адаптивная сетка «Последние обновления»

Дата: 13.07.2026

## Цель

На главной странице блок «Последние обновления» должен показывать пять карточек сериалов в строке на широком экране, не задавать карточкам принудительную одинаковую высоту и оставаться удобным на телефонах и планшетах.

## Решение

Изменение ограничивается Blade-разметкой блока в `resources/views/catalog/index.blade.php`. Данные, порядок тайтлов, компонент `x-title-card` и остальные постерные блоки не меняются.

Сетка получает явные адаптивные ступени:

- одна карточка до `sm`;
- две карточки на `sm`;
- три карточки на `md`;
- четыре карточки на `lg`;
- пять карточек на `xl` и более широких экранах.

У блока удаляется `auto-rows-fr`. Выравнивание элементов начинается от верхней границы, а применяемое внутри общего компонента карточки растягивание по высоте локально отменяется только для этой сетки. Благодаря этому высота каждой карточки определяется её содержимым, а другие места использования `x-title-card` сохраняют текущее поведение.

## Мобильное поведение

На телефоне карточка сохраняет существующую горизонтальную компоновку с постером слева. Начиная с `sm`, компонент продолжает использовать вертикальную карточку. Переходы сетки исключают горизонтальный скролл и не обрезают названия или метаданные.

## Проверка

Feature-тест главной страницы должен подтвердить наличие отдельного маркера блока, классов сетки `sm` / `md` / `lg` / `xl`, пяти колонок на `xl` и локальной отмены растягивания карточек. После изменения запускаются focused PHPUnit-тест и `npm run build`.

## Вне области изменения

- блок «Сейчас можно смотреть»;
- количество тайтлов, выбираемых для главной страницы;
- структура и данные карточки;
- футер и другие части макета.
