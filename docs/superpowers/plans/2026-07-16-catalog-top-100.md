# Публичные рейтинги «Топ 100» — план реализации

**Цель:** добавить четыре быстрые, индексируемые и адаптивные страницы рейтинга каталога с единым алгоритмом, существующими карточками и понятной точкой входа в навигации.

**Архитектура:** enum определяет четыре категории; query-service строит один публичный watchable ranking и загружает bounded карточки; page builder готовит вкладки, состояния и SEO; controller только выбирает view. Один Blade-шаблон разделяет первые три и места 4–100.

**Стек:** Laravel 13, PHP 8.5, Eloquent/SQLite, Blade, Tailwind CSS 4, PHPUnit 12, Playwright, Vite 8.
