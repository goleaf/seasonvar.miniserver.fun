# Правила проекта

## Контекст

Это Laravel-приложение для каталога сериалов Seasonvar.

- PHP 8.5
- Laravel 13
- Laravel Boost 2
- Laravel Pint 1
- PHPUnit 12
- Tailwind CSS 4

## Обязательные правила

- Перед изменениями в Laravel-коде использовать документацию Laravel Boost.
- Следовать текущей структуре каталогов и стилю соседних файлов.
- Не менять зависимости без явного запроса.
- Не добавлять рекламные, маркетинговые или заглушечные тексты на страницы.
- Весь видимый интерфейс писать на русском языке.
- Для PHP-правок запускать `vendor/bin/pint --dirty --format agent`.
- Для проверок использовать минимальные релевантные тесты.

## Импорт Seasonvar

- Основная команда: `php artisan seasonvar:full-sync`.
- Команда должна подробно показывать ход работы.
- Каталог должен сохранять сезоны и серии, а не показывать ложные нули.
- Видеоисточники Seasonvar, плеер, плейлисты и потоки не извлекать.

## Markdown

Корневые Markdown-файлы должны описывать этот проект, а не стандартный шаблон Laravel.
