# Правила проекта

## Контекст

Это Laravel-приложение для каталога проектов Seasonvar: сериалы, передачи, шоу, аниме, документальные страницы, сезоны, серии, связи каталога, отзывы и внешние видео-ссылки.

- PHP 8.5
- Laravel 13
- Laravel Boost 2
- Laravel Pint 1
- PHPUnit 12
- Tailwind CSS 4

## Обязательные правила

- Перед изменениями в Laravel-коде использовать документацию Laravel Boost.
- Следовать текущей структуре каталогов и стилю соседних файлов.
- Не менять зависимости без явного запроса.
- Не добавлять рекламные, маркетинговые или заглушечные тексты на страницы.
- Весь видимый интерфейс писать на русском языке.
- Для PHP-правок запускать `vendor/bin/pint --dirty --format agent`.
- Для проверок использовать минимальные релевантные тесты.

## Импорт Seasonvar

- Основная команда: `php artisan seasonvar:import`.
- Команда должна подробно показывать ход работы.
- Каталог должен сохранять сезоны и серии внутри одной карточки проекта, а не показывать отдельные сезоны как отдельные проекты.
- Видео не скачивается на сервер. Импорт сохраняет найденные внешние ссылки, качество, формат, перевод и состояние доступности.
- Старые медиа без стабильного ключа должны постепенно дополняться внутри `seasonvar:import`.
