# Каталог сериалов Seasonvar

Laravel-приложение для локального каталога сериалов, сезонов, серий, постеров, тегов и подключенных лицензированных медиа.

## Основные команды

```bash
php artisan seasonvar:full-sync
php artisan seasonvar:full-sync --once --discover=500 --parse=25
php artisan test --compact
vendor/bin/pint --dirty --format agent
```

## Импорт

Команда синхронизации скачивает карту сайта, сохраняет найденные страницы, разбирает карточки сериалов и обновляет сезоны с сериями. Серии берутся из данных страницы Seasonvar и сохраняются в таблицу `episodes`.

## Интерфейс

Интерфейс каталога ведется на русском языке. Нельзя добавлять на страницы рекламные или заглушечные описания проекта.

## Медиа

Видео подключается только из собственного разрешенного хранилища. Ссылки плеера, плейлистов и видеопотоков Seasonvar не извлекаются.
